## 인자, 매개변수 parameter

- 함수를 정의 할 때 외부로부터 받아들이는 임의의 값

```js
fuction fn (x, y) { /* do somtiong */};
// x, y를 인자 혹은 매개변수, parameter라고 한다.
```

##  인수 argument
- 함수를 호출할 때 이 때 사용하게 되는 일련의 값

```js
fn(3, 4);
// 3, 4를 인수, argument라고 한다.
```
