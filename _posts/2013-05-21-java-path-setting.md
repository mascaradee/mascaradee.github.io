## jdk_compiler 실행 환경변수 path 잡기

`cmd`(콘솔) 창에서 `javac` 명령어를 사용 하기 위해서는 `jdk`의 `bin`의 경로를 환경변수에 추가해 주면 된다.  

1. 내컴퓨터>우클릭>속성  

![path-setting1.jpg](/images/path-setting1.jpg)

2. 시스템변수

Path에 `C:\Program Files\Java\jdk1.7.0_21\bin`(저장된 경로에 따라 다름) 추가
