### 코파일럿 사용 방법과 단축키
 
 - 제안 창 열기: ctrl + enter
 - 코파일럿 코드 완성 제안 강제 실행: alt + \
 - 제안 선택: tab
 - 다른 제안 보기: alt + [ / alt + ]
 - 코파일럿 채팅 창 열기: ctrl + shift + i
