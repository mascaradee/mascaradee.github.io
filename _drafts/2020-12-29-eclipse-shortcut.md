## keys

- ctrl + shift + l :  단축키 모음 열기
- preference: alt + w -> p
- 타입 열기 : ctrl + shift + t
- Qiuck outline : ctrl + o

- ctrl + 1 : Quick Fix
- ctrl + 3 : find Actions 전체 검색
- ctrl + F8 : persfectives 탭 이동
- ctrl + F7 : views 탭 이동
- ctrl + F6 : editor 탭 이동


## Tips

자료형이 뭔지 모를때 임의의 자료형을 넣어놓고 메서드에 마우스를 가져다대면 적합한 자료형을 이클립스가 추천해 준다.   
이때 ctrl + 1을 눌러 빠르게 수정해 준다.
